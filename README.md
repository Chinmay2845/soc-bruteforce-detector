# soc-bruteforce-detector
Python-based SOC monitoring system for detecting SSH brute-force attacks through authentication log analysis, alert triaging, MITRE ATT&amp;CK mapping, and incident reporting.
Developed a Python-based SOC monitoring solution capable of detecting brute-force authentication attacks through real-time log analysis. Implemented automated incident generation, severity classification, MITRE ATT&CK mapping, JSON/CSV reporting
